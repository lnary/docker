# Gradle

Maven+Gradle的环境，maven源已配置使用阿里云镜像。

嗯，很多时候，使用Gradle的环境也是跟Maven密不可分的，所以就写了这个。

## 容器内的一些配置

Gradle本地库 `/home/gradle/.gradle`
Maven本地仓库挂载 `/var/maven/repository`
